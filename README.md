# Hospital-System
The Hospital System is a web-based application built on ASP .NET Core MVC that provides a platform to manage doctors. The application allows users to perform CRUD (create, read, update, delete) operations on doctors, and also allows viewing details of doctors. The UI has been designed to be user-friendly and responsive for a better user experience on different screen sizes.

# Features 
The Hospital System has the following features:

- Add new doctors to the system <br>
- Update existing doctors' information<br>
- Delete doctors from the system<br>
- View a list of all doctors<br>
- View details of a specific doctor<br>
- User-friendly UI for easy navigation and use<br>

# Installation
To install the Hospital System, follow the steps below:<br>

1- Clone the repository to your local machine using the following command:<br>

Copy<br>
git clone https://github.com/<username>/<repository-name>.git<br>
2- Open the solution file (HospitalSystem.sln) in Visual Studio.<br>

3- Build the solution to restore the NuGet packages and generate the required files.<br>

4- Run the application by pressing F5 or clicking on the "Start" button in the toolbar.<br>

5- The application should launch in your default browser.<br>

# Usage
To use the Hospital System, follow the steps below:<br>

1- Navigate to the "Doctors" section of the application.<br>
2- To add a new doctor, click on the "Add Doctor" button and fill in the required fields.<br>
3- To update an existing doctor's information, click on the "Edit" button next to the doctor's name and make the necessary changes.<br>
4- To view details of a specific doctor, click on the "Details" button next to the doctor's name.<br>
5- To delete a doctor from the system, click on the "Delete" button next to the doctor's name and confirm the action.<br>
